# UNISEX XL - Root Demos

This directory contains standalone demos for the UNISEX XL project.

## Quick Access

Browse all demos: [demos/index.html](./index.html)

## Directory Structure

```
demos/
├── ascii/              # ASCII art visualizers
│   └── ascii-visualizer-test.html
└── strudel/           # Strudel live coding demos
    ├── strudel.html
    └── strudel-webcam-ascii.html
```

## Demos

### ASCII Visualizers
- **[ascii-visualizer-test.html](ascii/ascii-visualizer-test.html)** - Test implementation of ASCII art visualization

### Strudel Live Coding
- **[strudel.html](strudel/strudel.html)** - Interactive live coding environment with Strudel
- **[strudel-webcam-ascii.html](strudel/strudel-webcam-ascii.html)** - Strudel + webcam ASCII visualization

## Navigation

- **Main App**: [../index.html](../index.html)
- **P5.js Demos**: [../p5js/demos/index.html](../p5js/demos/index.html)

## Running the Demos

1. Start the development server: `npm run dev`
2. Navigate to any demo file in your browser
3. Or use the index pages for easy browsing
